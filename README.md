# RobloxAsyms-Manual-Archipelago
A Manual for Roblox Asym games such as Forsaken, Die of Death, Outcome Memories, and Scream Jam

This archipelago is meant to be played by honoring the characters you have and only using them. The win condition is getting 20 wintokens (Which might be changed soon)

Items
- Survivors
- DoD movesets
- DoD Synergies
- Killers
- Skins as filler items for Forsaken, DoD, and SJ
- Reset trap
- Wintokens

Locations
- Challenges for each survivor and killer
- Optional Map wins
